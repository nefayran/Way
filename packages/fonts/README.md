# @waykit/fonts

Shared fonts for @way project and micro/monolithic frontend projects.

## Installation

### npm
```sh
npm install @waykit/fonts
```
### yarn
```sh
yarn add @waykit/fonts
```
## How to use
Add to your main.*:
```sh
import "@waykit/fonts";
```
and config @waykit/styles package for fonts.
