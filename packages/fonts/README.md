# @way/fonts

Shared fonts for @way project and micro/monolithic frontend projects.

## Installation

### npm
```sh
npm install @way/fonts
```
### yarn
```sh
yarn add @way/fonts
```
## How to use
Add to your main.*:
```sh
import "@way/fonts";
```
and config @way/styles package for fonts.
