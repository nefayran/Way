# @way/styles
## Styles library
Styles library include:
- Grid system
- Color system
- Typography
- Misc styles

## Installation

### npm
```sh
npm install @way/styles
```
### yarn
```sh
yarn add @way/styles
```
## How to use
Add to your main.*:
```sh
import "@way/styles";
```
