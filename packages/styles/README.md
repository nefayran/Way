# @waykit/styles
## Styles library
Styles library include:
- Grid system
- Color system
- Typography
- Misc styles

## Installation

### npm
```sh
npm install @waykit/styles
```
### yarn
```sh
yarn add @waykit/styles
```
## How to use
Add to your main.*:
```sh
import "@waykit/styles";
```
